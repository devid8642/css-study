Exemplo de flexbox 2

Abaixo temos uma div que está é um container e várias divs que são de filhos dessa div principal. Além disso, tem-se as propriedades relativas ao flexbox e alguns botões para alterar os valores dessas propriedades.

flex-direction:

justify-content:

align-items:

Com a propriedade flex-wrap é possível lidar com a largura da div e com um certo conteúdo dentro dela de forma automática. O conteúdo se distribuirá dentro da div sem extrapolar o perímetro da div e tentando o máximo possível manter o conteúdo no tamanho especificado.

Sem wrap. Os elementos abaixo foram dimunídos pois a propriedade flex-shrink de cada elemento filho de container está setada como 1.
