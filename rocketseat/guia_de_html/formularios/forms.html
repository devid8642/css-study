Entrada de dados com formulários

Com a tag input é possível especificar os campos de entrada de dados de um formulário. Segue os vários atributos suportados nesse elemento:

  • Type: Especifica o tipo do input
  • Autocomplete: Possibilia sugestões de preenchimento baseadas na memória do navegador.
  • Disable: Não permite o preenchimento do input.
  • Value: O input será carregado com um valor inicial.
  • Required: Não permitirá o envio do formulário sem o preenchimento dos inputs com esse atributo.
  • Placeholder: Permite a adiação de texto no fundo de um input para indicar o tipo de dado que deve ser preenchido.
